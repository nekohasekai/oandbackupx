Backup modes: An app can be backed up with only its apk, only its data or both. An app not currently installed can only be restored if the apk is present.

Borrar respaldo: Elimina los archivos del respaldo de una instancia específica. Mientras que "eliminar respaldos" elimina todos los respaldos de la aplicación elegida.

Desinstalar: Algo más agresivo que una desinstalación normal. Además de hacer una desinstalación normal a través de los comandos de Android, la desinstalación desde OAndBackupX elimina los archivos que la aplicación pueda haber dejado en /data/app-lib/.

Habilitar / deshabilitar: Utiliza el script de Android `pm` para habilitar o deshabilitar una aplicación. Deshabilitar una aplicación la elimina de la interfaz usual de usuario sin desinstalarla. Se puede utilizar para habilitar o deshabilitar una aplicación para varios usuarios a la vez (si el dispositivo tiene varios usuarios). Los usuarios se identifican con un ID, por ejemplo, 0 es el primer usuario (propietario), 10 suele ser el perfil de trabajo.

Múltiples usuarios: El multiusuario es todavía algo experimental pero debería funcionar. Habilitar y deshabilitar sólo funciona después de una instalación inicial (no necesariamente desde OAndBackupx) o la restauración de la aplicación.

Aplicaciones sin sistema: el respaldo y la restauración de las aplicaciones sin sistema instaladas como módulos Magisk no es compatible y el intento podría conducir a un fallo del sistema y obligar a formatear y flashear de nuevo.

Navegación: La barra de navegación tiene "Configuración", "Inicio", "Respaldo por lotes", "Restauración por lotes" y "Programar respaldos" (en ese orden). El número de aplicaciones actualizadas después de su último respaldo se presentará como una notificación en el botón "Inicio".

Aplicaciones que utilizan almacén de claves: Android tiene un almacenamiento cifrado para los datos sensibles. Las aplicaciones que hacen uso de esto para almacenar datos de acceso o algunas claves para su funcionamiento no pueden ser restauradas con éxito. Dos ejemplos de estas aplicaciones son Signal y Element(Riot).
